# **Entity-level Sentiment Classification**

在2023年秋季 舆情分析 综合大实验中，我们小组（sty、smy、lsm、ywt）实现了中、美、台三方视域下的舆情实体立场分析平台。

### 方法：

**基于多标签 BIO 标注方式的序列标注任务实现实体和立场联合识别。**



### 成果

- 包含==8000==条样本的高效可用数据集
- 一个在线Web应用平台
- 完整的实验报告
- 项目演示demo视频
- 开源前后端、AI模型所有代码
